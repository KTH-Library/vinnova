# Flattened complete data for calls, programmes, rounds and projects since a given date

Flattened complete data for calls, programmes, rounds and projects since
a given date

## Usage

``` r
vinnova_latest(from_date = Sys.Date() - 3)
```

## Arguments

- from_date:

  the date since changes are requested, Default: Sys.Date() - 3

## Value

object with tables

## Examples

``` r
if (FALSE) { # \dontrun{
if(interactive()){
 vinnova_latest()
 }
} # }
```
